HTML stands for HyperText Markup Language. It is the standard language used to create and structure web pages on the internet.HTML is called a markup language because it uses tags to mark up or label content.


HTML5 is the latest and improved version of HTML. Both are used to create web pages, but HTML5 includes new features, better multimedia support, and improved performance.


DOCTYPE html is a declaration that tells the web browser which version of HTML the page is using. In modern web development, it is used to tell the browser to render the page in HTML5


In HTML, tags and elements are closely related but have different meanings. A tag is the part written inside angle brackets, such as , and it is used to define the beginning or end of an HTML component. Tags are usually of two types: opening tags and closing tags. An element, on the other hand, is the complete structure that includes the opening tag, the content, and the closing tag.


Semantic tags in HTML are tags that clearly describe the meaning and purpose of the content they contain. These tags help browsers, developers, and search engines understand the structure of a webpage more easily. Semantic tags make the code more readable and improve accessibility and SEO


The main difference between semantic and non-semantic tags in HTML is that semantic tags clearly describe the meaning and purpose of the content, while non-semantic tags do not provide any information about the content they contain.


Meta tags are special HTML tags that provide information about a webpage to browsers and search engines. They are placed inside the head section of an HTML document and are not directly visible on the webpage. Meta tags help define details such as the page description, character encoding, author name, keywords, and viewport settings for responsive design.


The source tag is used inside audio and video tags to specify different media files and formats for the browser. Different browsers support different audio and video formats, so using multiple source tags helps ensure that the media can play correctly in most browsers. The browser automatically selects the first supported file format.


iframe (Inline Frame) is an HTML element used to display another webpage, document, video, map, or external content inside the current webpage. It allows one HTML page to be embedded within another HTML page.


Block-level elements and inline elements are two types of HTML elements that differ in how they display content on a webpage. Block-level elements always start on a new line and take up the full width available. They create a separate block of content, which means the next element appears on the next line. These elements are mainly used for larger sections or structure of a webpage. Examples of block-level elements are div, p, h1, section, and article.
