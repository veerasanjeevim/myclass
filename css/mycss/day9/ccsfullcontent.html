cascading specificity

In CSS cascading means that when multiple CSS rules apply to the same HTML element, the browser decides which rule wins based on a priority system.

1 st priority=inline

2nd priority=id selector

3rd priority=class selector

4rth priority=element selector

5th priority=universal element
