CSS (Cascading Style Sheets) is a styling language used to design and format web pages. It works with HTML to improve the appearance of a website. HTML creates the structure of a webpage, while CSS controls how it looks. Why is CSS used? CSS is used to make websites attractive and organized. It helps change colors, fonts, text size, spacing, layouts, and animations. It also makes websites responsive so they look good on mobile phones, tablets, and computers. CSS saves time because one style can be applied to many web pages, giving a consistent design throughout a website.


three type of css: internal external and inline


The difference between inline, internal, and external CSS is based on where the CSS code is written and how it is applied to a webpage. Inline CSS is written directly inside an HTML element using the style attribute, and it affects only that specific element. Internal CSS is written inside a tag within the of an document, and it is used to single webpage. External CSS is written in a separate .css file and connected to the HTML file using a tag, allowing the same styles to be used across multiple webpages.


In real-time projects, External CSS is mostly used because it keeps the code clean, organized, and reusable. Developers write CSS in a separate .css file and link it to HTML pages, making it easier to manage styles for large websites. If changes are needed, developers can update one CSS file and the changes will apply to multiple pages. Internal CSS is sometimes used for small projects or single pages, while Inline CSS is rarely used in real projects except for quick testing or applying a unique style to a specific element.


CSS selectors are used to select and style HTML elements on a webpage. They help apply CSS styles to specific elements such as headings, paragraphs, buttons, or divisions. A selector tells the browser which HTML element should receive a particular style. For example, you can use an element selector to style all paragraphs, a class selector to style a group of elements, or an ID selector to style one unique element.


The difference between class and ID selectors in CSS is that a class selector is used to style multiple elements, while an ID selector is used to style one unique element on a webpage. A class selector is written using a dot (.) before the class name and can be reused many times in HTML. An ID selector is written using a hash (#) before the ID name and should be used only once for a specific element. Also, an ID selector has higher priority than a class selector in CSS.


The universal selector in CSS is used to select and apply styles to all HTML elements on a webpage. It is represented using an asterisk (*) symbol. Developers commonly use it to apply common styles such as margin, padding, font, or box sizing to every element at once. This helps maintain consistent styling across the webpage and reduces repetitive code.


Cascading in CSS refers to the process the browser uses to decide which CSS style should be applied when multiple styles target the same HTML element. The browser follows a priority order to choose the final style. For example, if inline, internal, and external CSS all style the same element, CSS gives preference based on priority. This process is called “cascading” because styles flow and override one another according to rules.


Specificity in CSS is the rule that decides which CSS style will be applied when multiple selectors target the same HTML element. It works by giving different priority levels to selectors. An ID selector (#) has higher specificity than a class selector (.), and a class selector has higher specificity than an element selector. This helps the browser determine which style should override another.

Flexbox (Flexible Box Layout) is a CSS layout system used to arrange and align elements inside a container in an easy and flexible way. It helps developers create responsive layouts by controlling the position, spacing, alignment, and size of items without using complicated CSS. Flexbox is mainly used to align elements horizontally or vertically, create equal spacing, center content, and make webpage layouts responsive for different screen sizes.


The difference between justify-content and align-items in Flexbox is that justify-content is used to align items along the main axis (usually horizontal direction), while align-items is used to align items along the cross axis (usually vertical direction). These properties help position elements inside a flex container.
